在 Dify 中安全调用飞书企业级数据能力

MCP KEY 是访问飞书企业级 MCP 服务的授权凭证。
通过申请 KEY，你的 AI 工作流可以安全地查询多维表格、管理日历、检索通讯录等企业数据——同时确保每位用户只能访问自己权限范围内的信息。

为什么需要 KEY？
KEY 解决了 AI 工作流访问企业数据时的三个核心问题
🔒

企业数据需要受控访问

飞书多维表格、通讯录等包含敏感企业数据。MCP 服务使用「超级应用令牌（TAT）」调用飞书 API，可访问全量企业数据。KEY 控制哪些 AI 工作流有权调用这些能力，防止未授权访问。

👤

确保数据只给对的人

即使你的工作流有 KEY 授权，每次调用时系统仍会验证当前操作用户的飞书身份，通过前置权限校验和后置数据过滤，确保你只能看到自己有权访问的数据。

📊

审计追溯每一次调用

KEY 作为授权凭证，与审计日志绑定。每一次 MCP 调用都会记录谁在什么时间用了什么工具，确保企业数据的使用合规可追溯，出了问题也能精准定位。
