Dify 部署引导
✨ 方式一：在预装 MCP Server 中填入 KEY（推荐）
  1. 进入 Dify 空间 → 工具 → MCP，找到已预装的 TAT Server
  2. 在对应 Server 配置的 Header 中粘贴 KEY Value
  3. 保存后该 Server 即可正常使用
  4. UAT Server 无需 KEY，完成飞书 OAuth 授权后直接可用
📋 方式二：自定义 MCP Server 端点
  1. 进入 Dify 空间 → 工具 → MCP → 添加 MCP Server
  2. 填写自定义 Server URL（如私有化部署端点）
  3. 在 Header 中填入 KEY Value
