使用说明
📋

如何获取 KEY？

在 MCP KEY 管理平台申请并审批通过后，进入我的 KEY 详情页复制 KEY Value。

⚡

安装后会发生什么？

插件解析 KEY 的授权范围，自动在当前空间创建 MCP Server 连接。每个 Server 的认证 Header 会自动填入你的 KEY。
服务命名格式：mcp-{key_id}-{server_id}

🔄

KEY 重置或 scope 变更怎么办？

重新粘贴新 KEY 并点击安装，插件会通过 server_identifier 识别已有连接，自动更新而非重复创建。
